## Instalacion clsx
> permite combinar clases condicionalmente
```bash
pnpm add clsx
```

## Instalacion zustand
> contexto global de la aplicacion
```bash
pnpm add zustand
```

## Instalacion prisma con Postgres
```bash
pnpm add prisma tsx --save-dev
pnpm add @prisma/extension-accelerate @prisma/client
```

## Inicializar prisma
```bash
npx prisma init

```
